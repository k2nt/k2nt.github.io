CarbonEdge Helm Chart Repository

This is the official Helm chart repository for CarbonEdge.

Usage

helm repo add carbonedge https://k2nt.github.io/helm/carbonedge
helm install carbonedge-tier2 carbonedge/sinfonia

Available Charts

  • carbonedge-operator-0.1.0.tgz

Ensure that your index.yaml is present in this same directory.
